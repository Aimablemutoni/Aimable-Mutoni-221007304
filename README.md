# Personal Web Projects

This repository contains four academic web projects built using HTML, CSS, JavaScript, PHP, and MySQL.

## Projects

### project1/

A simple website layout developed according to the assignment requirements. It includes basic navigation and structured content sections.

### project2/

A student registration system with a form connected to a MySQL database using PHP scripts for data handling and storage.

### project3/

A multi-page hotel website featuring an order form, contact form, user authentication (login/logout), and database integration using PHP.

### project4/

A currency converter application built with external CSS for styling and JavaScript for functionality.

## Technologies Used

* HTML
* CSS
* JavaScript
* PHP
* MySQL

## Purpose

These projects were developed for academic purposes to demonstrate foundational web development skills and basic full-stack integration.
